About Typing Test Pro

Features

  • Real-time WPM and accuracy tracking
  • Multiple time duration options
  • Challenge mode with progressive difficulty
  • Global leaderboard
  • Beautiful, responsive design

About the Developers

This web project has been developed by Garvit and Karan, students currently pursuing their Bachelor of Computer Applications (BCA) in the 6th semester at Soban Singh Jeena University (SSJU), Almora.

With a passion for web technologies and software development, they collaborated on this project as part of their academic journey, applying theoretical knowledge to real-world problem-solving.

Technologies Used

  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • Backend: Node.js with Express
  • Deployment: Vercel
