About Typing Test Pro

Features

  • Practice Mode: Improve your typing speed with various time options
  • Game Mode: Engage in fun typing challenges with progressive difficulty
  • Real-time WPM and accuracy tracking
  • Global leaderboard to compete with others
  • Beautiful, responsive design with modern UI effects

About the Developers

This web project has been developed by Garvit and Karan, students currently pursuing their Bachelor of Computer Applications (BCA) in the 6th semester at Soban Singh Jeena University (SSJU), Almora.

With a passion for web technologies and software development, they collaborated on this project as part of their academic journey, applying theoretical knowledge to real-world problem-solving. Their goal is to continue growing as full-stack developers and contribute to impactful digital solutions.

Technologies Used

  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • Backend: Node.js with Express
  • Deployment: Vercel
